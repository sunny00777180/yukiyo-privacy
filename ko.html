개인정보처리방침

JittaKey – 일본어 입력 키보드

최종 업데이트: 2025년 6월 18일

1. 수집하는 개인정보 항목

이 앱은 사용자가 입력한 내용, 키보드 입력 기록, 연락처, 위치정보 등의 개인정보를 수집하지 않습니다.
앱 내 설정 정보(알파벳/스테가나 표시 ON/OFF)만 사용자의 단말기에 저장되며 외부 전송은 없습니다.

2. 전체 접근 허용 요청

YUKIYO 키보드는 타이핑 소리 및 진동 피드백 기능을 위해 “전체 접근 허용”을 요청할 수 있습니다.
그러나 입력 내용은 수집하거나 외부로 전송하지 않습니다.

3. 제3자 제공

개인정보를 제3자에게 제공하거나 외부 서버로 전송하지 않습니다.

4. 보유 및 삭제

YUKIYO는 별도의 서버 저장 없이 앱 내 설정 정보를 단말기에만 저장하며, 앱 삭제 시 해당 정보도 함께 삭제됩니다.

5. 이용자의 권리

이용자는 언제든지 앱을 삭제하거나 초기화할 수 있으며, 문의 사항은 아래 이메일로 연락 바랍니다.
